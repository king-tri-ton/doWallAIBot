# Telegram-бот для генерации изображений по тексту с API OpenAI

Этот бот позволяет создавать изображения на основе текстовых описаний с помощью OpenAI. Вы отправляете текст, бот генерирует изображение и присылает обратно.

---

## Установка и настройка

1. **Склонируйте репозиторий:**

```
git clone https://github.com/king-tri-ton/doWallAIBot.git
cd doWallAIBot
```

2. **Создайте виртуальное окружение (рекомендуется):**

```
python -m venv venv
source venv/bin/activate    # Linux / Mac
venv\Scripts\activate       # Windows
```

3. **Установите зависимости:**

```
pip install -r requirements.txt
```

4. **Создайте файл `.env`:**

* Скопируйте `.env.example` в `.env`:

```
cp .env.example .env        # Linux / Mac
copy .env.example .env      # Windows
```

* Заполните свои данные в `.env`:

```
BOT_TOKEN=ваш_токен_бота
AI_TOKEN=ваш_API_ключ_OpenAI
ADMIN_ID=ваш_ID_администратора
IMAGE_FOLDER=папка_для_изображений  # например images
```

5. **Запуск бота:**

```
python bot.py
```

---

## Использование

* **/start** — начать работу с ботом.
* **Отправка текста** — после команды `/start` отправьте текстовое описание изображения.
* Бот вернёт сгенерированное изображение в виде файла.
* **/stat** — получить статистику (только для администратора).

---

## Важные замечания

* Убедитесь, что зависимости установлены и `.env` заполнен корректно.
* Использование OpenAI API может быть платным в зависимости от тарифа.
* Бот предоставляется «как есть», использование — на ваш страх и риск.
